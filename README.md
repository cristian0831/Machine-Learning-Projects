# Machine Learning Projects

### This set of projects are based on data extracted from Kaggle, a community of data scientists that allow find and publish a set of data to create predictive models. Most of this proyects is saved in the directory Scikit-Learn. The directory's name is because I work mainly in projects oriented in Machine Learning applications by using the famous python library Scikit-Learn.

#### There are additional projects that are based on the notebook containing the solution to the exercises of the O'Reilly book  Hands-on Machine Learning with Scikit-Learn, Keras and TensorFlow.

### Additional problems outside the Scikit-learn are based on the projects solutions of the CS50’s Introduction to Artificial Intelligence with Python course.
